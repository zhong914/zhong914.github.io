ss极简易配置

发表于 2017-03-24

  • 首先新建AMI一切免费的选项

  • 获取root权限
    sudo passwd root

  • 切换root身份
    su root
  • AMI自带pip,直接装shadowsocks
    pip install shadowsocks
  • 新建配置文件夹
    mkdir /etc/shadowsocks
  • 新建配置文件
    sudo vim /etc/shadowsocks/config.json

  • 编辑文件

  • 最简单的方式跑起来,因为我是免费帐号,需要在不同国家中切换,所以随用随开
    cd /etc/shadowsocks/
    ssserver

  • 更新实例的入站规则,添加自定义tcp,端口为你刚刚开ss的如8388
  • 完,参考http://blog.csdn.net/gsying1474/article/details/25306129.https://my.oschina.net/lanni654321/blog/489459
